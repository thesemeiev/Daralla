Что такое VPN

VPN направляет трафик через удалённую ноду выхода и помогает стабильнее работать в разных сетевых условиях. Сервис упрощает подключение, смену серверов и управление доступом из одного приложения.

Почему мы

  • Стабильное соединение клиент ↔ узел выхода
  • Несколько транспортных профилей для разных сетевых условий
  • Минимизация телеметрии: без хранения истории посещённых ресурсов
  • Единая подписка на пул серверных нод без ручной миграции ключей
  • Две схемы аутентификации: веб-кабинет и Telegram Mini App

Пакеты трафика по нодам, назначение нод на пакеты и квота оплаченного периода — на вкладке «Трафик».

Автоматическая синхронизация

При сохранении изменений подписка автоматически синхронизируется со всеми X-UI серверами. Поля этой вкладки (название, срок, лимит устройств) применяются на всех серверах; лимиты и учёт трафика настраиваются отдельно на вкладке «Трафик».

Удаление подписки

Внимание! Это действие необратимо.
Подписка будет полностью удалена из базы данных, а все клиенты удалены со всех X-UI серверов.

Лимиты трафика по нодам

Пакеты — какая нода подписки на каком лимите (безлимитный или лимитный пакет). Включённый трафик на оплаченный период и докупка считаются в блоке «Квота оплаченного периода» (обновление после оплат; тариф задаёт множитель месяцев). С панелей списание идёт сначала из включённого объёма, затем из докупки. Каждая нода назначается ровно на один пакет; при исчерпании лимита клиент на этой ноде отключается в X-UI.

Назначение нод на пакеты

Для каждой строки выберите пакет; затем нажмите «Сохранить назначения» (отдельно от кнопок в карточках пакетов).

Загрузка...

Тарифы

Используются в мини-приложении и на экране оплаты. У каждого тарифа — свой ключ периода (например month), название для пользователя, длительность в днях, цена и опциональный бейдж.

Докупка трафика

Пакеты для пользователей с лимитным учётом по группе (есть строка квоты на подписке). Ключ пакета — латиница, цифры, _ и -. Объём — двоичные ГиБ (1024³). Цена списывается только из этого списка при оплате.
